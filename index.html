Featured Work

PoisonSpot

PoisonSpot is my flagship machine learning security project. It is a defense system that protects machine learning models from stealthy clean-label data poisoning attacks by tracing how training samples influence model parameter updates.

The system outperformed the state-of-the-art defense ASSET on benchmark datasets including CIFAR-10, STL-10, and ImageNet-100, and was released as an open-source framework with full experimental pipelines.

The artifact received ACM CCS 2025 badges for Artifacts Available, Artifacts Evaluated – Functional, and Results Reproduced.
